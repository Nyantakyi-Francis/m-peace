Module 4

Assessing Learners’ Speaking and Listening Skills

A Six-Module Professional Learning Package by Miss Mprah Ruhaimatu

1. Module Overview

This module develops student teachers’ understanding of how to assess oral language development effectively and fairly. It focuses on oral language assessment, pronunciation assessment, listening comprehension assessment, and common challenges in evaluating speaking and listening. It is based on the fourth unit of the course outline.

2. Learning Outcomes

By the end of this module, student teachers should be able to:

  1. Explain the purpose of assessing speaking and listening.
  2. Identify tools and methods for oral language assessment.
  3. Assess pronunciation, comprehension, and oral performance.
  4. Address challenges in assessing diverse learners.

3. Module Content

Why Assessment Matters

Assessment helps teachers to monitor progress, identify learning gaps, guide instruction, give feedback, and support learner development.

What Can Be Assessed in Speaking?

Teachers can assess: pronunciation, grammar, vocabulary, fluency, clarity, confidence, interaction skills.

What Can Be Assessed in Listening?

Teachers can assess: attention, comprehension, interpretation, response, recall, note-taking, ability to follow instructions.

Methods of Assessing Speaking

Examples: oral presentations, storytelling, interviews, role-play, dialogue performance, group discussion, picture description.

Methods of Assessing Listening

Examples: listen-and-answer tasks, dictation, follow-the-instruction tasks, note-taking tasks, retelling tasks, oral response tasks.

Assessment Tools

Useful tools include: checklists, rating scales, rubrics, anecdotal notes, observation sheets, peer assessment forms.

Assessing Pronunciation

Teachers may assess sound production, stress, intonation, clarity, intelligibility.

Important caution: Assess clarity, not accent discrimination.

Problems in Assessing Speaking and Listening

Common challenges: subjectivity, large classes, limited time, learner anxiety, multilingual classrooms, inconsistent scoring.

Making Assessment Fair

Assessment should be clear, criterion-based, inclusive, supportive, continuous, and aligned to learning outcomes.

Good Oral Assessment Practice

Good oral assessment should be frequent, authentic, allow rehearsal, include feedback, and value improvement, not perfection.

4. Interactive Learning Activities

  • Activity 1: Rubric Design
    Groups create a rubric for storytelling, oral presentation, or listening comprehension.
  • Activity 2: Assessment Observation
    Students watch/listen to a short oral task and score it using a checklist.
  • Activity 3: Bias Reflection
    Prompt: “How can oral assessment disadvantage some learners if done poorly?”

5. Assessment Tasks

Assignment

Develop a speaking or listening assessment tool for JHS learners and explain how it supports fairness and diverse learners.
